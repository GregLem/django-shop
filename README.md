# 🛒 Django Shop — интернет-магазин с API

Полноценный интернет-магазин на Django с корзиной, админкой и REST API.

## 🚀 Возможности

- 📋 **Каталог товаров** с пагинацией (12 товаров на странице)
- 🗂️ **Категории** с подсчётом количества товаров
- 🛒 **Корзина** для гостей (сессии) и авторизованных пользователей
- 👤 **Регистрация и вход** с защищёнными страницами
- 🛡️ **Админка** для управления товарами и категориями
- 📡 **REST API** с фильтрацией, поиском и сортировкой
- 🎨 **Адаптивная вёрстка** с CSS-переменными, Flexbox и Grid
- 💰 **Фиксация цены** в корзине (цена не меняется при обновлении товара)

## 🛠️ Стек технологий

| Технология | Назначение |
|------------|------------|
| Django 5.2 | Веб-фреймворк |
| Django REST Framework | REST API |
| django-filter | Фильтрация в API |
| SQLite | База данных |
| HTML/CSS | Вёрстка (Flexbox, Grid, CSS-переменные) |

## 📡 API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/products/` | Список товаров |
| POST | `/api/products/` | Создать товар |
| GET | `/api/products/{id}/` | Детали товара |
| PUT | `/api/products/{id}/` | Обновить товар |
| DELETE | `/api/products/{id}/` | Удалить товар |

**Фильтрация:** `?category=1` | `?search=Мурзилка` | `?ordering=-price`

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/GregLem/django-shop.git
cd django-shop

# Установить зависимости
pip install -r requirements.txt

# Применить миграции
python my/manage.py migrate

# Создать суперпользователя
python my/manage.py createsuperuser

# Запустить сервер
python my/manage.py runserver
