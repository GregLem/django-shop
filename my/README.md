# 🛒 Интернет-магазин на Django

## Что умеет
- Каталог товаров с пагинацией
- Корзина (гость + пользователь)
- Регистрация и вход
- Админка для управления товарами
- REST API с фильтрацией и поиском
- Адаптивная вёрстка

## Стек
- Django 5.2
- Django REST Framework
- SQLite
- HTML/CSS (Flexbox, Grid, CSS-переменные)

## Запуск
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
